O que é HTML?

O HTML (HyperText Markup Language), criado por Tim Berners-Lee nos primórdios da internet, pode ser considerada a linguagem mais básica para a construção de uma página web, pois ela é responsável por definir o conteúdo da mesma, o acrônimo HTML traduzido para o português ficaria algo como "Linguagem de marcação de hipertexto".

A linguagem funciona por meio dos chamados elementos que definem o tipo de conteúdo sendo exibido e, usando as tags que formam o elemento, marcamos onde ele começa e termina. Por exemplo, podemos usar a tag <p> para começar um parágrafo e </p> para terminá-lo.

Mais adiante veremos mais detalhadamente sobre os elementos HTML.

Outra característica fundamental é a presença dos hyperlinks ou apenas links, que são textos clicáveis que criam conexões entre as páginas (basicamente um arquivo .html) de um mesmo site (conjunto de páginas) ou entre sites diferentes, uma grande facilidade e inovação da época em que o HTML foi criado.

Os hiperlinks e a função da linguagem de "marcar" o conteúdo são suas principais características e dão à linguagem seu nome.

Por ser bastante básica, HTML é comumente usada com a linguagem CSS para deixar a página visualmente agradável e com JavaScript para a interatividade com o usuário.

Mesmo simples, é a mais primordial das 3 linguagens, visto que não há o que estilizar nem o que interagir se não tiver primeiro o conteúdo, que é estabelecido por meio do HTML.

"Eu só tive que pegar a ideia de hipertexto e conectá-la às ideias de TCP e DNS e — ta-da! — a World Wide Web."

Tim Berners-Lee

Os elementos do HTML

Como apresentado anteriormente, os elementos do HTML são usados para marcar o conteúdo e delimitá-lo. Alguns exemplos de elementos:

  • <h1>
  • <strong>
  • <a>
  • <p>

Um parágrafo escrito em HTML ficaria da seguinte forma:

<p>Eu sou um parágrafo</p>

Dois tipos de elemento

Geralmente, os elementos são formados por uma tag de abertura (<p> ) e uma de fechamento (</p> ) que consiste na mesma tag porém com uma barra à frente do nome do elemento.

Porém, alguns elementos não precisam de uma tag de fechamento, pois eles não pedem um "conteúdo" a ser adicionado para funcionarem, como uma imagem ou um campo de texto. Exemplos de elementos que não precisam de uma tag de fechamento:

  • <input>
  • <br>
  • <hr>
  • <img>

Um campo de texto escrito em HTML ficaria simplesmente:

<input />

Note que a barra já está presente no que seria a "tag de abertura" visto que não é necessária uma tag de fechamento.

Elementos semânticos

Com o lançamento o HTML5, sua versão mais recente até o momento, foi introduzida a semântica à linguagem, isto é, elementos que carregam consigo um significado. Esses elementos quando bem utilizados deixam a página mais acessível para pessoas com algum tipo de deficiência e também ajudam a melhorar o SEO do site. Exemplos de elementos semânticos:

  • <section>
  • <header>
  • <nav>
  • <article>

Esses elementos servem para indicar o que é, por exemplo, uma sessão de uma página ou talvez um cabeçalho.

Mesmo não tendo impacto visual, isso ajuda muito para que leitores de tela (programas que ajudam pessoas cegas a navegarem na internet) possam funcionar da melhor forma.

Atributos

Para alterar o funcionamento padrão de um elemento, pode ser usado um atributo dentro dele para isso.

Alguns atributos são globais (podem ser usados em qualquer elemento) e outros são específicos e muitas vezes até obrigatórios de certos elementos.

Exemplos de atributos:

  • id
  • href
  • lang
  • alt
  • style

O atributo "lang", por exemplo, define qual a linguagem do conteúdo apresentado pelo elemento, na prática ele seria aplicado assim:

<p lang='en'>I am a paragraph</p>

O atributo é inserido dentro da tag de abertura do elemento.

No exemplo acima, o atributo "lang" recebeu o argumento "en", indicando que aquele parágrafo está escrito em inglês, o que ajuda bastante na acessibilidade.

Entendido como funcionam elementos e atributos, podemos focar em como funciona o HTML e alguns elementos específicos.

O elemento de imagem

Para exibir uma imagem na tela, é utilizada a tag <img /> que, como pode ter notado, é o tipo de tag que se fecha sozinha.

Contudo, para que o computador saiba que imagem exibir, você precisa indicar o caminho ou link da imagem em questão.

Isso é feito por meio do atributo "src", que recebe como argumento o link da imagem. Exemplo:

<img src='(insira aqui o link ou caminho da imagem)'/>

Esse é o tipo de atributo obrigatório para uma tag img, pois sem ele nada será exibido.

Atributo "alt"

Outro atributo que pode ser usado numa tag img é o "alt", que seria o texto alternativo, isto é, uma breve descrição da sua imagem.

É muito importante usar o alt por questões de acessibilidade, pois o leitor de tela vai ler a descrição da imagem para o usuário. Ele não terá nenhum impacto visual.

O texto alternativo somente será exibido no lugar da imagem caso ela não apareça por algum motivo, como conexão ruim ou link quebrado.

Os links ou âncoras

No começo você leu que uma das principais características do HTML são os links, também chamados de hiperlinks ou âncoras

Agora que sabemos sobre o funcionamento dos elementos e seus atributos, podemos ver como um link funciona.

A tag usada para criar um link é <a>, ela precisa de um conteúdo então é necessária a utilização da tag de fechamento (</a>)

Entre as tags, você vai digitar o texto que será usado como link.

Exemplo:

<a>Eu sou um link!</a>

Porém esse link precisa ir para algum lugar, certo? É aí que entra o atributo "href", ele receberá como argumento o link propriamente dito, ou seja, o caminho que aquele texto levará quem clicar nele.

Exemplo:

<a href='exemplo.html'>Eu sou um link! Clique em mim"</a>

Esse código exibirá:

Eu sou um link! Clique em mim

Esse é um link real feito com o mesmo código que foi dado de exemplo! Pode clicar!

Note que o link tem uma estilização própria do navegador, ele é azul e possui sublinhado. Você pode alterar isso com CSS mais tarde.

O atributo "target"

Se você clicou no link dado de exemplo, você deve ter sido apresentado ao atributo "target", que fez outra página ser aberta no seu navegador, ele não é obrigatório como o "href" mas é indispensável se você se importa com a expêriencia do usuário do seu site.

Para utilizá-lo, basta adicionar o atributo com o argumento "_blank", se o que você deseja é que o link abra em outra página.

Exemplo:

<a href='https//www.youtube.com/' target="_blank">Esse link leva ao YouTube (sem te tirar do meu site)"</a>

Irá exibir:

Esse link leva ao YouTube (sem te tirar do meu site)

Misturando elementos

Existem algumas tags voltadas para texto e estilização, como é o caso da <b>, que serve para deixar o texto em negrito

Mas talvez você tenha se perguntado, "e se eu quiser deixar apenas uma palavra em negrito dentro de um parágrafo? Você pode! Basta adicionar um elemento dentro de outro.

Exemplo:

<p>Eu <b>amo</b> HTML.</p>

Esse código exibirá:

Eu amo HTML.

Você pode fazer o mesmo usando links e outros elementos que vimos.

As listas

Listas são elementos HTML um pouco mais complexos, pois precisam de dois elementos distintos, um dentro do outro.

Exemplo de lista:

<ul>
 <li>Arroz</li>
 <li>Óleo</li>
 <li>Bananas</li>
 <li>Açúcar</li>
</ul>

Isso exibirá:

  • Arroz
  • Óleo
  • Bananas
  • Açúcar

A lista é composta pelo elemento <ul> com elementos <li> dentro dela, criando cada um dos itens.

Listas ordenadas x Listas não-ordenadas

O que você viu de exemplo é chamada de lista não-ordenada, isto é, uma lista em que a ordem não realmente importa e cada item é marcado por um ponto.

O outro tipo de lista que temos no HTML como você pode imaginar é a lista ordenada, nela, a ordem dos itens importa e eles receberão um número em vez de um ponto.

Exemplo de lista ordenada:

<ol>
 <li>Palmeiras</li>
 <li>Grêmio</li>
 <li>AM</li>
 <li>Flamengo</li>
</ol>

Isso exibirá:

  1. Palmeiras
  2. Grêmio
  3. AM
  4. Flamengo

Note que a única coisa que mudou foi o elemento "ul" que virou "ol", que significa ordered list, ou seja, lista ordenada, o contrário de unordered list, que é lista não-ordenada.

Títulos

Outro elemento essencial para qualquer página são os títulos ou headings.

Para criar um título, usamos a tag <h1>.

No caso, essa tag criaria um título de nível 1, isto é, o título de maior importância na sua página.

Existem 6 níveis diferentes de título (h1 ao h6), sendo que geralmente só existe um h1 em um página e os outros são subtítulos, subtítulos dos subtítulos e assim por diante.

Exemplo:

Título 1 (h1)

Título 2 (h2)

Título 3 (h3)

Título 4 (h4)

Título 5 (h5)
Título 6 (h6)

Vale ressaltar que você usa os títulos conforme o grau de importância deles, não por questões de estética de tamanho , visto que isso pode ser alterado com CSS.
